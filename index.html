01

Website MLX CAPITAL

Projeto Confidencial. Ainda não pode ser divulgado o link do projeto no entanto é um projeto freelancer frontend de um website de um cliente da AUCLAN DESIGN que fornece serviços de investimentos de capital na área da saúde.

Um projeto desenvolvido em html e css puro com algumas automações em javascript usando conceitos de pixel perfect e design minimalista.

Pixel Perfect refere-se à prática de criar um design que corresponde exatamente ao mockup ou ao design original, pixel por pixel. Isso é especialmente importante para designers e desenvolvedores que querem assegurar que o produto final na web seja uma réplica fiel do design conceitual. Este conceito pode envolver o uso de grid systems, unidades relativas como rem , e ferramentas de design para comparar o site com o mockup

O design minimalista foca na simplicidade e na funcionalidade. Este estilo geralmente utiliza uma paleta de cores limitada, tipografia clara e funcional, espaços em branco (white space) para criar foco no conteúdo principal, e evita elementos desnecessários. A ideia é que "menos é mais", proporcionando uma experiência visual limpa e direta.
